#  Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML, CSS, and JavaScript**.  
This project showcases my skills, projects, services, and contact information in a clean and professional layout.

##  Live Demo
You can deploy the website using GitHub Pages or any hosting platform.

*(https://philglenny24.github.io/my-portfolio/)*

##  Features

- Responsive design for all screen sizes
- Modern UI with clean layout
- Multiple sections:
  - Hero section
  - About section
  - Services section
  - Projects section
  - Skills section
  - Contact section
- Grid and Flexbox layout (no CSS frameworks used)
- Interactive navigation menu (mobile-friendly)
- Smooth scrolling between sections
- Clean typography and color scheme

##  Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (basic interactivity)

## Design Principles

- Uses a dark theme with complementary accent colors
- Consistent spacing and alignment
- Card-based layout for services, projects, and skills
- Responsive grid system for scalability

## Responsiveness

The website is fully responsive and adapts to:
- Mobile devices
- Tablets
- Desktop screens

##  How to Run Locally

1. Clone the repository:

git clone https://github.com/philglenny24/my-portfolio.git

2. Navigate to the project folder:

3. Open `index.html` in your browser:
- Double click the file OR
- Use Live Server (VS Code extension)

## Deployment (GitHub Pages)

1. Push your project to GitHub
2. Go to repository settings
3. Navigate to **Pages**
4. Select:
- Source: `Deploy from branch`
- Branch: `main`
5. Save and wait for the link to generate
